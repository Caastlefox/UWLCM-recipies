Scripts may contain user-specific file and folder paths, and grant names, that need to be modified. Non-MPI builds use UWLCM Singularity images that can be downloaded from: https://zenodo.org/records/15630519. Alternatively, they can be built using instructions from https://zenodo.org/records/15630519. MPI builds require module installation by admins (already done on Athena) or Spack package installation (Fugaku)

Scripts are available for following clusters:
- Cyfronet Athena (Slurm)
- RIKEN Fugaku (PJM, Spack)

Model scaling tests:
- Cyfronet Prometheus (https://doi.org/10.5194/gmd-15-4489-2022)
- Fugaku (https://doi.org/10.5281/zenodo.20185835)
